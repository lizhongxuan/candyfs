# CandyFS

CandyFS is a distributed file system that provides high availability and performance.

## Usage

```bash
# Show version
candyfs version

# Start the service
candyfs start

# Stop the service
candyfs stop

# Check status
candyfs status
```

## Building and Installation

```bash
# Build the project
go build -o candyfs

# Install the binary
go install
```
